How We Built This Site

This website is built with HTML, CSS, and JavaScript and no frameworks or libraries. We wanted to show we understand how websites actually work from the ground up.

The responsive design usesCSS to make everything look good on phones, tablets, and computers. All of the interactive features, like the search filters on the resources page and the form on the submit page use custom JavaScript we wrote ourselves.

We picked a purple and blue color scheme because blue makes people feel trust and purple adds creativity. All the gradients, card animations, and hover effects are done with CSS. We tested the colors to make sure they have good contrast so everyone can read the text easily.

The resource search system works by using JavaScript to filter through all the community organizations in real-time. When you type in the search box, it instantly shows matching results without reloading the page.

Accessibility & Performance

We made sure the site is accessible and fast:

  • High contrast colors that meet accessibility standards
  • Keyboard navigation works on all buttons and links
  • Mobile-friendly design that works on any screen size
  • Fast loading because we didn't use heavy frameworks
